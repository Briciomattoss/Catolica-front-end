Linguagens de programação
Tipos de linguagem Descrição
Linguagens Web HTML Não é uma linguagem de programação, mas sim uma linguagem de marcação.
JavaScript É uma linguagem de programação amplamente utilizada para desenvolvimento web.
PHP Linguagem de programação voltada ao desenvolvimento de páginas dinâmicas.
Linguagens Desktop C Linguagem de baixo nível usada em desenvolvimento de sistemas operacionais e drivers.
C++ Versão melhorada da linguagem C, com suporte à programação orientada a objetos.
Java Usada em diversos tipos de aplicações, desde desktop até sistemas embarcados.

Framework Descrição

Angular Framework mantido pelo Google para aplicações web dinâmicas.

Bancos de Dados Populares
Bancos de Dados Relacionais e Não-Relacionais
MySQL Banco de dados relacional muito usado em aplicações web. Gratuito e de código aberto. Usado com PHP e WordPress.
PostgreSQL Avançado e robusto, com suporte a tipos de dados complexos. Ideal para aplicações que exigem integridade e desempenho.
MongoDB Banco NoSQL baseado em documentos (JSON). Escalável e muito utilizado em aplicações modernas.
